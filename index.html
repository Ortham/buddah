What Is Buddah?

Modding communities are hotbeds of creativity, and community software often allows mod authors to go well beyond what's possible with official modding tools, if they exist at all.

However, it can be quite difficult to find out what software has already been written. Between mods that use the same terminology, and other software that might be unrelated but use similar names, looking for something in the relatively small niche of game modding software is like looking for a needle in a haystack.

Buddah aims to help software developers avoid reinventing the wheel by providing a registry for software that is specific to game modding. Using it, you can see what already exists, and break new ground instead!

Buddah is made by WrinklyNinja using Bootstrap and Lunr.js, and hosted on GitHub. If you like Buddah, please consider donating.

Buddah is named after a member of the TES IV: Oblivion modding community who maintained a list of mod links containing over 16,000 entries, to help others find what they sought. It's not a spelling mistake. ;)
