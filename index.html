ES Modding Tools

A simple registry of third-party modding tools and libraries for Bethesda's Elder Scrolls and Fallout games.

This site is a GitHub repository. Please fork it and add any missing projects you know of, and spread the word! Any utility can be added, on GitHub or not, open source or not, and in any state of completion. The more we know about what's already done, the less work we may end up repeating.
